Code Style — общие правила написания кода

Чтобы коллегам по проекту было проще разобраться в вашем коде, он должен быть не только свободным от «костылей», но ещё читабельным и однообразным.
В этом помогает кодстайл — общие правила написания кода.

Code Style HTML

Теги

Обычные теги. Теги оформляются строчными буквами. Давайте посмотрим на примерах.

Некорректно:

В примере выше теги div и Img прописаны с заглавных букв — такое написание не соответствует общим правилам, а читать код становится сложнее. Нужно опустить всё в нижний регистр.

Оформление классов
